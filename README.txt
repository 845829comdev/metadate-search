MetaDate-JOOT
=============

Простая но мощная программа для извлечения метаданных из изображений.

Запуск:
1. Установи библиотеки: pip install -r requirements.txt
2. Запусти: python main.py

Особенности:
- Извлекает ВСЕ возможные EXIF данные
- Поддержка RAW форматов (CR2, NEF, ARW)
- GPS координаты с конвертацией
- Экспорт в JSON
- Простой интерфейс

Всего 4 файла - максимум функционала!
